# IUT Localisation

## Mobile Application for Campus Geolocation

This mobile application aims to provide geolocation services specifically for campus navigation. Using a combination of GPS and user input, the app offers:

- **Real-time location tracking**
- **Indoor mapping capabilities**
- **Route optimization**
- **Accessibility features**

Created with user experience in mind, the application strives to enhance the way students navigate the campus, making it easier to find classrooms, libraries, and other important locations.

### Features
- User-friendly interface
- Offline maps availability
- Multi-platform support (iOS and Android)

### Getting Started
1. Download the application from the app store.
2. Enable location services on your device.
3. Follow the on-screen instructions to begin navigating the campus.
