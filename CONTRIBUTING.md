# Contributing to IUT Localisation

Thank you for considering contributing to the IUT Localisation project! We appreciate your efforts to help improve the project. Here are some guidelines to help streamline the contribution process:

## How to Contribute

1. **Fork the Repository**  
   Start by forking the repository to your own GitHub account.

2. **Create a New Branch**  
   It's best practice to create a new branch for each feature or bug fix. You can create a new branch using the following command:
   ```bash
   git checkout -b your-feature-branch
   ```

3. **Make Your Changes**  
   Implement your changes and make sure to write clear, concise commit messages that explain your reasoning for the changes.

4. **Push Your Changes**  
   Push the changes to your forked repository:
   ```bash
   git push origin your-feature-branch
   ```

5. **Create a Pull Request**  
   Go to the original repository and click on the **Pull Requests** tab, then click **New Pull Request**. Select your branch and submit your pull request. Include a description of your changes and any related issues.

## Code of Conduct

Please ensure that your contributions align with our project’s [Code of Conduct](CODE_OF_CONDUCT.md).

## Issues

If you find any bugs or have feature requests, please open an issue on the GitHub repository.

## Thank You!

Thank you for your interest in contributing to the IUT Localisation project! Your contribution helps us improve and grow!  

Happy coding!
